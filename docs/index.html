Umbra Framework

Umbra is an HTML5 game framework created for the JS13KGames Game Jam, in which the total size of your submission is limited to only 13KB. Umbra is ultra-lightweight, with a specialized tool designed to cut out the parts that you don't need in your project. A minimal installation is only 431B!

For a guide on getting started with Umbra and Umbra Builder, see the sections below.

Getting Started

To get started with Umbra, download Umbra Builder and walk through its prompts. An in-depth guide on how to use the Umbra Builder is located in the documentation if you need it. Also in the documentation is a description of each tag, and an example project for after you've built your Umbra distribution.
